Help for Folder Browser

  • Select a Folder. Select a folder and enter it as a command-line argument:
    java -cp classes ser321.FolderBrowser aFolderPath
    If you don't enter an argument, then the Browser assumes the current working directory.
  • Browser another Folder. Enter an absolute or relative path to another folder in the text field at the top of the browser. Then click the button to rebuild the tree. Since the program builds the entire tree, its probably best to not select a folder with lots of files and sub-directories.
  • Select Files. Expand sub-trees and select terminal nodes. When you select a file with one of these file types, its contents are displayed in the right panel: txt, java, cpp, hpp, html, xml.
